===========
ARISTOXENUS
===========

Who
    I am a hobby programmer who started learning Python in February of 2023. 

What
    Aristoxenus is a musical manipulation tool. This doc is mostly just a placeholder for now while I keep working on the program. Here are a few things it can do so far:

    best_heptatonic_scale
        Choose the best name for a given interval structure and tonic note name.
    force_heptatonic_scale
        Force a given heptatonic interval structure to follow ABCDEFG nomenclature regardless of any accidentals.
    parse_chord_symbol
        Attempt to generate an interval map from the sub-symbols in a given chord symbol.
    encode_scientific / decode_scientific
        Used to translate scientific notes (e.g. A4) to frequencies (e.g. 440 hz) and vice versa.

In the near future, it will also be able to:

- provide chord and scale names for given interval structures (including modes/inversions)
- parse complex polychord symbols into interval structures
- parse slash chord symbols into interval structures

A few goals of the program for the more distant future include:

- developing a system of rhythmic notation that's easily readable for humans and python parse_chord_symbol
- learning how to use frequency data to generate MIDI files from given pitch data
- simulate the specific range of real instruments
- predict plausible fingerings for piano and guitar chords
- create tablature for tin whistle, recorder, and other aerophones

As I keep learning more about programming, long term aspirations might be:

- creating a decision-making hierarchy to help generate legitimate-sounding melodic material
- creating a procedural permutator to generate infinite etudes based on simple structures and chord progressions
- render musical data in sheet music format


Pitch Structures
----------------
Pitch structures are stored as integers, allowing us to manipulate
musical material using bitwise operations.

Nomenclature
------------
Musical nomenclature is automatically generated by the program.

Octave
------
The octave is the smallest theoretical pitch structure.

Double Octave
-------------
The double octave is the largest theoretical pitch structure.

Instruments
-----------
The real range of an instrument can be represented by a multi-octave pitch structure.
